#  Sales Performance Dashboard | Kenyan Retail Sales | Power BI Project

## Project Overview

This project is an interactive **Sales Performance Dashboard** developed in **Power BI** to analyze retail sales performance across different regions, cities, product categories, payment methods, and sales representatives in Kenya.

The purpose of this dashboard is to showcase my practical skills in:

* Business Intelligence Reporting
* Data Modeling
* Data Analysis Expressions (DAX)
* KPI Development
* Data Visualization
* Business Performance Analysis
* Dashboard Design and Storytelling

The dashboard provides insights that can support strategic decision-making, sales optimization, profitability analysis, and operational monitoring.


# Dashboard Preview

The dashboard contents: 

* Sales KPIs
* Regional performance analysis
* Sales representative performance
* Category-level analysis
* Monthly sales trends
* Payment method distribution
* Interactive filtering.


# Business Objectives

The primary objective of this project was to design a Business Intelligence dashboard capable of helping retail stakeholders:

1. Monitor overall sales performance.
2. Track profitability and profit margins.
3. Identify high-performing regions and cities.
4. Analyze product category performance.
5. Evaluate sales representative contributions.
6. Understand monthly sales trends and seasonality.
7. Assess customer payment preferences.
8. Enable data-driven business decisions through interactive reporting.


## Dataset Information

This is a simulated Kenyan retail sales dataset containing transactional sales records from multiple regions and cities across Kenya.
It is an AI generated dataset that replicates the retail industry in Kenya. 


The dataset contains **400 sales transactions** and was structured to support Business Intelligence reporting and analytical modeling.


# Dataset Columns and Description

| Column Name      | Description                                                      |
| ---------------- | ---------------------------------------------------------------- |
| `order_id`       | Unique identifier for each sales order                           |
| `order_date`     | Date the transaction occurred                                    |
| `month`          | Month of the transaction                                         |
| `quarter`        | Fiscal quarter of the transaction                                |
| `customer_name`  | Name of the customer                                             |
| `city`           | City where the transaction occurred                              |
| `region`         | Kenyan region associated with the transaction                    |
| `product_id`     | Unique identifier for the product                                |
| `product_name`   | Name of the product sold                                         |
| `category`       | Product category (Electronics, Furniture, Stationery, Groceries) |
| `quantity`       | Number of units purchased                                        |
| `unit_price`     | Price per unit of the product                                    |
| `total_sales`    | Total revenue generated from the transaction                     |
| `cost`           | Cost incurred for the transaction                                |
| `profit`         | Profit earned from the transaction                               |
| `payment_method` | Customer payment method used                                     |
| `sales_rep`      | Sales representative responsible for the sale                    |

# Key Insights from the Dashboard

## 1. Central Region Generated the Highest Sales

The Central region recorded the highest sales contribution, outperforming other regions.

This suggests:

* Strong customer demand
* Better market penetration
* Higher purchasing power in the region


## 2. Electronics Was the Best-Performing Category

Electronics generated the highest sales revenue among all categories.

This indicates:

* High consumer demand for tech products
* Greater revenue contribution per transaction
* Potential opportunities for inventory expansion



## 3. Strong Profitability Performance

The business achieved a profit margin of approximately 39%, indicating healthy operational profitability.


## 4. Mobile Payments Dominated Transactions

M-Pesa and Airtel Money accounted for a significant portion of customer payments.

This reflects:

* High mobile money adoption in Kenya
* Importance of digital payment integration
* Customer preference for convenient payment methods


## 5. Sales Performance Fluctuated Across Months

The monthly trend analysis revealed noticeable variations in sales performance throughout the year.

This could indicate:

* Seasonal purchasing behavior
* Promotional effects
* Demand fluctuations across different months

## 6. Sales Representatives Showed Varied Performance Levels

The sales representative analysis highlighted differences in:

* Revenue generation
* Number of orders handled
* Profit margins achieved

This insight can support:

* Sales coaching
* Incentive planning
* Workforce performance management
